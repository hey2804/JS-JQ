用 axios 串接 氣象局api


右鍵inspect，可以看到consle.log有印出資料。
axios 其實就是個物件，
Q: 我們要取物件的値該怎麼取？
A: 會用 物件名. 屬性 去取得値
axios .get 不外乎就是 一個叫 axios 的物件有一個叫 get 的屬性
    axios優點:
  • 使用方法類似於jQuery
  • 可以搭配 async/await
  • 支持Promise
  • 可支援 Node.js 從後端發送的 Http request，這意味著 axios 可以兼用於前端與後端專案。
  • 支援防CSRF
  • 輕量化，約13KB
  • 可以取消要求
  • 直接自動將回應的 JSON 資料轉換成 JavaScript 的 Object，這十分方便！
  • 可以用 axios.create 方法做API domain的管理(RESTful api???)
axios缺點:
需要使用catch做錯誤的捕捉與處理。
